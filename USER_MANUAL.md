# 鶯谷杯 小倉2026 - ユーザーマニュアル（クイックリファレンス）

このアプリは、競馬の戦績（投資額・回収額）をリアルタイムで共有し、ランキングを競うためのツールです。

## 📱 基本的な使い方

### 1. ログイン / 初回登録
画面左上の **人間のアイコン**（または「ログイン」ボタン）からログイン画面へ移動します。

1. **ユーザー選択**: 自分の名前を選びます。
2. **初回登録**:
   - 初期パスワードは全員 `0000` です。
   - 初めてログインするときは、**新しい4ケタの数字（PINコード）** を設定してください。
3. **2回目以降**: 設定したPINコードを入力してログインします。
   - 名前（馬名）はここから自由に変更できます。

### 2. 戦績を報告する
ログイン後、画面右下の **「＋」ボタン** を押して入力を開始します。

1. **レース選択**: 開催場（小倉/東京/阪神）とレース番号（1R〜12R）を選びます。
   - ⚠️ **注意**: 発走時刻前のレースは選択できません（ロックされています）。
2. **金額入力**:
   - **投資額**: そのレースに使った金額（円）
   - **回収額**: 払い戻された金額（円）
3. **「報告する」** を押して完了！

### 3. 修正する
間違えて入力した場合、ランキングリストの自分の名前をクリック（またはタップ）すると修正画面が開きます。

---

## 🏆 ランキング・アイコンの見方

ランキングは **「回収率（％）」** の高い順で決まります。

| アイコン/項目 | 説明 |
| :--- | :--- |
| **順位** | 回収率（回収額 ÷ 投資額）に基づいた現在の順位です。 |
| 👑 (王冠) | 現在の **1位** のユーザーです。 |
| ⭐ (星) | **投資王**（総投資額が最も多いユーザー）です。 |
| **金文字** | プラス収支（勝ち）の状態です。 |
| **白文字** | マイナス収支またはトントンの状態です。 |

## ❓ よくある質問

**Q. PINコードを忘れました**
A. 管理者（拓也さん）に連絡してリセットしてもらってください。

**Q. 「未発走」となって入力できない**
A. 不正防止のため、実際の発走時刻を過ぎるまで入力できないようになっています。

**Q. 自分の名前がない**
A. 参加者リストは固定されています。追加が必要な場合は開発者へ連絡してください。
